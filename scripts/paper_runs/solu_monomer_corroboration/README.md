# Solu_monomer structural-context corroboration

Independent 5-enzyme corroboration of the §3.3 CATH structural-context finding
(Ensemble broadens candidate-pool spread + sequence diversity vs the single
backbone).

**Update 2026-06-18:** the 3-arm selected-set version (single+surrogate /
RFD3+BioEmu pool / RFD3+BioEmu+surrogate) is now **included in the manuscript** as
the independent real-enzyme corroboration in Supplementary Note 13, with a
one-sentence pointer from §3.3. Produced by `monomer_threeway.py`; the
distribution figure (box-and-whisker + per-enzyme points) is generated by
`../make_threeway_distribution_figures.py`. The earlier optA/optB 2-arm scripts
below are retained as the original pool-level check.

## Targets (Solu_pipeline_benchmark enzymes)
1ATJ, 1CQW, 1LVM, 1TCA, 5XJH (tier 50% conservation)

## Conditions
- **Original**: single (target) backbone.
  - v1: legacy `abl_be_<t>_single_s1` only (af2 ~10/target)
  - v2: legacy `abl_be_<t>_single_s1` + `solu_monomer_optB_<t>_single_s2` (af2 ~16-20/target)
- **Ensemble**: `abl_be_<t>_{bioemu,rfd3_single,rfd3_bioemu}_s1` combined (af2 ~30/target)

## Result (both v1 and v2)
All 5/5 targets: Ensemble > Original on pLDDT range, SoluProt range, and mean
pairwise diversity. Paired Wilcoxon exact p = 0.0625 (theoretical minimum for n=5,
i.e. unanimous direction). Conclusion unchanged after adding Option B Original
designs (v2), which strengthens robustness.

## Caveat
- `diversity` (mean pairwise identity) is sample-size robust → trustworthy.
- `range` metrics are extremum-based and partly confounded by pool size
  (Original af2 16-20 / designs 52 vs Ensemble af2 30 / designs 121).
- Direction is consistent and corroborates §3.3; treat as supporting evidence,
  not a standalone claim.
