Usage Guide

          
        

        

          

            
Quick Start

            
    
              
  • 
                  Advanced: choose Pipeline or Workflow Studio, attach target input, run preflight, then launch.
              
    • 
              
  • 
                Monitor: follow stage status, review checkpoint artifacts, and continue from the monitor panel when needed.
              
    • 
              
  • 
                Analyze: start with Compare Studio, Run-to-Run Compare, and Hit List, then save feedback, experiments, and reports.
              
    • 
            

          

          

            
Advanced Tips

            
    
              
  • 
                Select a recent run and use Load Request to copy its settings into a new run before editing.
              
    • 
              
  • 
                Use Resume Run in Monitor or Studio for same-run continuation. Advanced is for fresh runs only.
              
    • 
            

          

          

            
Monitoring

            
    
              
  • 
                Select a recent run to load status, artifacts, reports, and workflow review context.
              
    • 
              
  • 
                Use Auto Poll for live updates and workflow actions when a checkpoint pauses execution.
              
    • 
            

          

          

            
Analysis

            
    
              
  • 
                Use Compare Studio for tier-based WT/RFD3/BioEmu comparison, and open folded baselines only when needed.
              
    • 
              
  • 
                Read the inline Comparison Summary, then review full-run deltas and ranked candidates in Run-to-Run Compare and Hit List.
              
    • 
              
  • 
                After triage, log feedback and experiments, then generate the final report package.
              
    • 
            

          

          

            
Admin

            
    
              
  • Admins can create users from the Admin button.
